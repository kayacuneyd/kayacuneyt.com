CKa Contact Form is a simple, user-friendly, and responsive contact form plugin for WordPress.

== Description ==

CKa Contact Form allows your visitors to send messages directly to your chosen email address through a simple and clean form. It's easy to set up and offers customizable email settings through the WordPress admin panel.

== Installation ==

1. Upload `cka-contact-form.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings -> CKa Contact Form to configure the plugin.

== Frequently Asked Questions ==

= Can I change the email address where the form sends messages? =

Yes, you can change it in the plugin settings under the WordPress admin panel.

= Is the form responsive? =

Yes, the form is designed to be responsive and works well on all devices.
